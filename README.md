# 🍔 Smart Food Stall Pre-Ordering System

A Flask-based web application that allows students to pre-order food, select preferred break time slots, and reduce crowd congestion at campus food stalls.

## 🚀 Features

### 👩‍🎓 Student
- Register / Login
- Browse Menu
- Add to Cart
- Select Time Slot
- Track Order Status
- View Recommended Least Crowded Slot

### 👨‍🍳 Admin
- Manage Menu Items
- Toggle Availability
- Track Peak Time Slots
- Update Order Status
- Demand Analytics

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML + Bootstrap
- Jinja2

## 📊 System Highlights
- Real-time demand tracking
- Slot capacity optimization
- Peak hour detection
- Crowd congestion reduction
